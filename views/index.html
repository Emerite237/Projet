bienvenue dans mon microservice qui sert effectue le CRUD sur les sites touristique

http://localhost:8080/api/findall/post pour afficher toutes les publications http://localhost:8080/api/findall/post?titre=element a rechercher pour des recherches plus precise http://localhost:8080/api/post pour cree un publication http://localhost:8080/api/post/modifier/:id pour modifier un publication http://localhost:8080/api/post/supprimer/:id pour supprimer une publication
